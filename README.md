# Web Development Portfolio Workshop

In this workshop we build a simple developer portfolio using:

- HTML
- CSS
- JavaScript

By the end you will understand how these technologies work together to build interactive websites.

---

## Workshop Structure

1. HTML – Create the structure
2. CSS – Style the website
3. JavaScript – Add interactivity
4. Deploy the website

## Requirements

1. VS Code
2. Git
3. Web Browser
