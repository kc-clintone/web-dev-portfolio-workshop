# Workshop Guides

These are the steps we will follow to build the website:


```
├── step-1-html
│   └── index.html   # Create the index page and add code
│
├── step-2-css
│   ├── index.html    # Create and link your external CSS file so that styles apply
│   └── style.css     # Add relevant styles to style what you choose
│
├── step-3-javascript # Create and link the javascript file
│   ├── index.html    # Update the index page and link the Js file
│   ├── style.css     # Add more styles (go crazy)
│   └── script.js     # Yes, you can add more Js code
```
